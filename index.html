Ciclo for en Java

El ciclo for en Java se compone de cuatro secciones, a continuación hablamos de ellas de acuerdo a su orden de ejecución:

  1. Inicialización: Aquí inicializamos una variable de tipo numérico (int normalmente) que nos servirá para avanzar por el ciclo, normalmente se usa el 0, pues la mayoria de veces se recorren arreglos o listas desde el primer item (desde el índice 0). Este paso se realiza una sola vez en todo el ciclo.
  2. Condición: Esta parte del ciclo donde se determina si debe seguir o no con la ejecución del ciclo. Aquí podemos usar los operadores lógicos (&&, ||) y los operadores de comparación numéricos (<,<=,>,>=) y los operadores de comparación universales (==, !=, !). Este paso se realiza antes de ejecutar el código dentro del ciclo.
  3. Código a ejecutar: Es precisamente el código que queremos ejecutar, es ese tipo de código que se debería ejecutar de forma repetitiva, y este ciclo ofrece una solución a ese problema. Este paso se ejecuta siempre y cuando la condición se haya cumplido.
  4. Incremento/decremento: De acuerdo a la forma en que queramos recorrer el ciclo, vamos incrementando o cecrementando nuestra variable inicializada. Normalmente se usa el ++ o el -- para afectar incrementar o decrementar de a 1, pero es posible agregar nuestra lógica. Este paso se realiza luego de ejecutar nuestro código dentro del ciclo.
Fuente de consulta de operadores de comparación 
for (Inicialización; Condición; Incremento/decremento)
{
    // Código a ejecutar
}

A continuación podrás ejecutar un ciclo de prueba para que comprendas más su funcionamiento. 

for (int contador = 0; contador < 5; contador++)
{
    System.out.println("El contador tiene: " + contador);
}
